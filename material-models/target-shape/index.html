    Target Shape

    During simulation, the Target Shape creates a force to pull the simulation object towards the Target Shape. It designed for achieving art directed shape. The strength of the pulling force is controlled Target Stiffness and Target Damping parameter. Target Shape needs to have the same number of vertices as the simulation object.
